PSFP Stream Gate Configuration Overview Help

This page allows the user to inspect the current PSFP stream gate configurations, along with options to add, edit or delete such configurationas.

Stream Gate ID
The ID of this stream gate instance.
Enabled
Indicates whether the stream gate is active (Enabled = Yes) or inactive (Enabled = No).
Gate State
The administrative value of the GateStates parameter for the stream gate.
Cycle Time
The administrative value of the cycle time for the gate in either milliseconds, microseconds, or nanoseconds.
Cycle Time Extension
The administrative value of the cycle time extension parameter for the gate in either milliseconds, microseconds, or nanoseconds.
Base Time Seconds/ISO 8601
The administrative value of the BaseTime parameter for the gate.
The "Seconds" value indicates the time in seconds and subseconds since epoch where the configuration is configured to change - if the Config Change flag has been issued.
The "ISO 8601" value represents the same, but is displayed as a string formatted according to ISO 8601.
IPV
The administrative value of the IPV parameter for the gate.
Control List Length
The number of entries in the Gate Control List.
Close Gate Due To Invalid Rx
If checked, the stream gate gets permanently closed if receiving a frame during a closed gate state.
Close Gate Due To Octets Exceeded
If checked, the stream gate gets permanently closed if receiving a frame that exceeds the configured Octet Max.

Configuration Buttons

On the right hand side of each stream gate, you find two buttons:
: Click to edit this stream gate.
: Click to delete this stream gate (happens immediately without confirmation).

The very last row in the table also contains a button:
: Click to create a new stream gate.

Buttons

: Click to refresh the page immediately. This is useful if the stream gate configuration has been changed from e.g. CLI.
