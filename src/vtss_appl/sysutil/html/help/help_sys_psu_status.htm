System Power Supply Status Help

The switch system power supply status is provided here.

ID
The identification of power supply.
Description
The description of power supply.
State
The description of power supply. Possible values are:
Active: The power supply is used for the system currently.
Standby: The power supply is standby as the redundant power supply.
Not Present: The power supply is not present.
