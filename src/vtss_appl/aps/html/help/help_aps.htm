APS Help

The Automatic (Linear) Protection Switch instances are configured here.

Delete
This box is used to mark an APS for deletion in next Save operation.
APS ID
The ID of the APS. Click on the ID of an APS to enter the configuration page. The range is 1-100.
Architecture

Port: This will create a 1+1 APS.

Port: This will create a 1:1 APS.

Working Port
The working port for the APS.
Protect Port
The protecting port for the APS.
Working SF MEP
The working Signal Fail reporting MEP.
Protect SF MEP
The protecting Signal Fail reporting MEP.
APS MEP
The APS PDU handling MEP.
Alarm
There is an active alarm on the APS.

Buttons

: Click to add a new APS entry.

: Click to refresh the page immediately.

: Click to save changes.

: Click to undo any changes made locally and revert to previously saved values.
