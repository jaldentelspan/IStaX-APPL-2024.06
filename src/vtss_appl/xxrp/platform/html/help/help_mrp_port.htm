MRP Overall Port Configuration Help

This page allows you to configure the MRP generic settings for all switch ports.

The settings relate to the currently selected stack unit, as reflected by the page header.

The displayed settings are:

Port
The port number for which the following configuration applies.
Join Timeout
Controls the timeout of the Join Timer for all MRP Applications on this switch port. This value is restricted to ?-? centiseconds.
Leave Timeout
Controls the timeout of the Leave Timer for all MRP Applications on this switch port. This value is restricted to ?- ? centiseconds.
LeaveAll Timeout
Controls the timeout of the LeaveAll Timer for all MRP Applications on this switch port. This value is restricted to ?- ? centiseconds.
Periodic Transmission
Enable or disable the PeriodicTransmission feature for all MRP Applications on this switch port.
